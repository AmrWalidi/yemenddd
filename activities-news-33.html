منح تخفيضات في جامعة السودان المفتوحة عن طريق جمعيةالصداقة

في إطار التفاهم والشراكة بين جامعة السودان المفتوحة والتي تعد إحدى الجامعات الحكومية السودانية المعترف بها لدى العديد من الجامعات الأوربية والعربية وجمعية الصداقة والتعاون اليمنية ـ تركيا منحت الجامعة تخفيضاً للطلاب اليمنيين في تركيا في جميع المراحل. علماً أن الدراسة في الجامعة تتم بنظام التعليم عن بعد وتتم المناقشة والامتحانات في تركيا عبر فرع الجامعة بإسطنبول، والتخفيض يشمل جميع التخصصات على النحو التالي:

البكالوريوس 600 دولار للعام الدراسي
الدبلوم العالي 1500 دولار الماجستير 3000 دولار
الدكتوراه 4500 دولار

التخصصات المتاحة:
1) الدراسات الإسلامية واللغة العربية ( بكالوريوس + ماجستير + دكتوراه )
2) اللغة العربية والتاريخ (بكالوريوس)
3) الدراسات الإسلامية والجغرافيا (بكالوريوس)
4) الدراسات الإسلامية والتاريخ (بكالوريوس)
5) الجغرافيا والتاريخ ( بكالوريوس + دبلوم عالي + ماجستير + دكتوراه )
6) الرياضيات والحاسوب (بكالوريوس)
7) الرياضيات والفيزياء (بكالوريوس)
8) الكيمياء والأحياء (بكالوريوس)
9) الأحياء والإنتاج الحيواني (بكالوريوس)
10) اللغة الإنجليزية (بكالوريوس)
11) رياض أطفال (بكالوريوس)
12) إدارة أعمال ( بكالوريوس + دبلوم عالي + ماجستير + دكتوراه )
13) المحاسبة (بكالوريوس + دبلوم عالي + ماجستير + دكتوراه )
14) القانون ( بكالوريوس + ماجستير + دكتوراه )
15) اللغات ـ اللغة العربية ــ اللغة الإنجليزية ( بكالوريوس + دبلوم عالي + ماجستير + دكتوراه )
16) الاقتصاد والعلوم المالية والإسلامية (إقتصاد إسلامي ـ تمويل إسلامي )
17) الإعلام ـ إذاعة وتلفزيون ــ العلاقات العامة والإعلان ـ الصحافة والنشر
18) التربية (طرق ومناهج التدريس) ( دبلوم عالي + ماجستير + دكتوراه)
19) اللغة العربية للنطاقين بغيرها ( دبلوم عالي + ماجستير + دكتوراه)
20) علوم حاسوب ( دبلوم عالي + ماجستير + دكتوراه)
21) تقنية معلومات ( دبلوم عالي + ماجستير + دكتوراه)
22) هندسة البرمجيات ( دبلوم عالي + ماجستير + دكتوراه)
23) السياحة والإرشاد الأثري ( دبلوم عالي + ماجستير + دكتوراه)
24) التراث الشعبي السوداني ـ الفلكلورـ ( دبلوم عالي + ماجستير + دكتوراه)
25) الأثار الإسلامية ( دبلوم عالي + ماجستير + دكتوراه)
26) الإعلام ( ماجستير + دكتوراه )
27) علم النفس ( ماجستير + دكتوراه )
28) الاقتصاد ( ماجستير + دكتوراه )
29) الترجمة ( ماجستير + دكتوراه )
30) الدراسات السودانية المصرية القديمة ( ماجستير + دكتوراه )
31) إدارة الأعمال الاحترافي MBA ( ماجستير )
